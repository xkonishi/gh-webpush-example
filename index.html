How to

  1. Open Push Companion Site. (Keep open it.)
  2. Copy & Paste Public Key.
  3. Click [ENABLE PUSH MESSAGING] button.
  4. Set Permit Nortification to agree if you need.
  5. Set a below json text to 'Subscription to Send To' field.
  6. Set a push message to 'Text to Send' field.
  7. Click [SEND PUSH MESSAGE] button.
